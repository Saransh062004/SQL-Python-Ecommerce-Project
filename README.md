
# SQL-Python-Ecommerce-Project

## 📊 Project Overview
This project demonstrates how to use Python and SQL to perform data analysis on an E-commerce dataset. The goal is to clean, process, and extract meaningful insights from the dataset using SQL queries executed through Python.

## 🛠️ Technologies Used
- Python
- SQLite
- Pandas
- Jupyter Notebook

## 📂 Project Structure
- `Questions.txt` – List of analytical questions to be answered.
- `csv_to_sql.py` – Python script to load CSV data into a SQL database.
- `dataset_link.txt` – Link to the dataset used.
- `python+sql_ecommerce.ipynb` – Jupyter Notebook containing all the analysis and SQL queries.

## 🚀 Features
- Data loading from CSV to SQL.
- SQL queries for key business metrics.
- Python-Pandas integration for post-processing.
- Clean and reproducible Jupyter Notebook workflow.
